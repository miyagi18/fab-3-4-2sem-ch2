## Практика 3. JSON + API + Postman

### Цель (что нужно уметь по итогам)

1. Понимать, что такое **JSON** и где он используется в API.
2. Уметь отправлять HTTP‑запросы через **Postman** (или альтернативно через `api.http`).
3. Протестировать:

- **свой API** (`/api/products`) — минимум 3 запроса,
- **внешний API** — минимум 5 запросов.

Требование по внешнему API и количеству запросов — см. методичку преподавателя.

---

### Часть A — запускаем backend

1. Откройте папку `backend/`.
2. Установите зависимости:

```bash
npm i
```

3. Запустите сервер:

```bash
npm run dev
```

4. Проверьте, что сервер отвечает:

- В браузере: `http://localhost:3000/`
- Или запросом в Postman: `GET http://localhost:3000/`

---

### Часть B — Postman: быстрый чек-лист

> Если Postman не получается — используйте файл `backend/api.http` через VS Code + расширение **REST Client** (по согласованию с преподавателем).

1. Откройте Postman → **New** → **HTTP Request**
2. Создайте запрос `GET http://localhost:3000/api/products` → нажмите **Send**
3. Создайте запрос `POST http://localhost:3000/api/products`
   - вкладка **Body** → **raw** → **JSON**
   - вставьте пример JSON (можно свой):

```json
{
  "title": "Печенье",
  "category": "Сладости",
  "description": "Тестовый товар",
  "price": 79,
  "stock": 10
}
```

4. Создайте запрос `PATCH http://localhost:3000/api/products/<id>` (id возьмите из ответа GET)
5. Создайте запрос `DELETE http://localhost:3000/api/products/<id>`

---

### Часть C — внешнее API (минимум 5 запросов)

Сделайте **5 запросов** к внешнему API и сохраните результаты (скриншоты).

Подсказка: пример внешнего API есть в `backend/api.http` (JSONPlaceholder, OpenWeatherMap и т.п.).  
**Важно:** если внешнее API требует ключ — получите ключ и храните его локально (в `.env`, не коммитьте в git).

---

### Что сдаём по практике 3

- Ссылка на репозиторий.
- Скриншоты выполненных запросов (можно в папку `docs/screens/`).
- Минимум:
  - 3 запроса к своему API,
  - 5 запросов к внешнему API.

## Полезные подсказки

- Если в браузере ошибка CORS — проверьте `backend/app.js` (origin).
- Если `req.body` пустой — проверьте `app.use(express.json())`.
- Если axios ругается на `TODO` — вы не реализовали функции в `productsApi.js`.
